5 comandos para la terminal

Corresponden a 5 comandos basicos para la terminal de linux.

  1. whoami = Muestra el nombre del usuario actual.
  2. ls = Muestra un listado con los archivos del directorio actual.
  3. touch = Crear un archivo, dandole una ruta o nombre.
  4. cd = Nos permite navegar por los directorios, hacia atras, adelante, etc.
  5. pwd = Muestra el directorio acutal, en el que estamos.

2 comandos 'Alias' preferidos

  1. Alias 'so' nos permite actualizar la shell (en mi caso zsh), en nuestra terminal

    2. alias so = "source ~/.zshrc"

  2. Alias 'cl' este alias nos limpia la terminal/shell, nos ahorra solo 3 letras, pero durante el dia son minutos.

    3. alias cl = "clear"
